First meeting

Pertama, kita belajar cara menggunakan Git, melalui cmd dan bash. Git sendiri merupakan version control system yang digunakan para developer untuk mengembangkan software secara bersama-bersama. Fungsi utama git yaitu mengatur versi dari source code program anda dengan mengasih tanda baris dan code mana yang ditambah atau diganti.

Second Meeting

Kedua, kita belajar cara menggunakan jQuery Mobile. Jquery adalah library atau kumpulan <--more--> kode javascript siap pakai. Keunggulan menggunakan jquery dibandingkan dengan javascript standar adalah, yaitu menyederhanakan kode javascript dengan cara memanggil fungsi-fungsi yang disediakan oleh jquery.

Third Meeting

Keitga, kita belajar menggunakan AngularJS. AgularJS berfungsi sebagai salah satu freamework yang paling populer dan banyak digunakan oleh developer untk mengembangkan aplikasi.
